# To-Do App

A simple to-do list application built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by All, Active, or Completed
- Clear completed tasks
- Persistent storage using `localStorage`
- Responsive layout with custom styled checkboxes

## Files
- `index.html` - app structure and layout
- `style.css` - app styling and custom checkbox design
- `script.js` - task logic, filtering, and persistence

## Instructions
1. Open `index.html` in a browser.
2. Enter a task in the input field and click the plus button or press Enter.
3. Use the filter buttons to view all, active, or completed tasks.
4. Click the task checkbox to mark it complete.
5. Click the `Clear Completed` button to remove completed tasks.

## Notes
- Tasks are saved in the browser so they persist across refreshes.
- The app uses `localStorage` to store tasks locally.
