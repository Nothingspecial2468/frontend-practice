# Form Validator

A responsive form validation application built using HTML, CSS, and JavaScript which validates user inputs in real-time and provides visual feedback for errors and successful submissions.

# 🚀 Features

- Real-time input validation
- Error and success indicators
- Custom success modal
- Animated success feedback
- Form reset after successful registration

# 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

# 🧠 Concepts Practiced

- DOM Manipulation
- Event Handling
- Form Validation
- Dynamic Class Management
- Custom Modal Implementation
- Css Animations
- Application Flow Design

# 🎉 Custom Enhancement

Instead of using the default browser alert, this project includes a custom success modal with:

- Celebration emoji
- Success message
- Overlay background
- Popup animation
- Manual dismissal through an action button
