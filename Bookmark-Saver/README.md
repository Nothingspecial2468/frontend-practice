# Bookmark Saver

A lightweight bookmark app built with vanilla JS that allows users to save, store, and manage useful links directly in the browser.


## 🚀 Features

- Save bookmarks instantly 
- Persistent storage using localStorage
- Dynamic UI rendering
- Delete bookmarks
- Clean interface


## 🛠 Tech Stack

- HTML
- CSS
- JavaScript


## 🧠 Behind The Scenes

This project focuses on understanding how data flows through an application:
- capturing user input
- storing data in objects
- syncing UI with stored data
- rebuilding interface dynamically from localStorage


## Concepts practiced

- DOM manipulation
- Event handling
- localStorage workflow and persistence
- Dynamic rendering
- Objects
- App flow logic
