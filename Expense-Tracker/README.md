# Expense Tracker

- A simple expense tracker application which helps user in tracking their income, expenses and total balance.

## 🚀 Features

- Add income and expenses transactions
- Dynamic balance calculations
- Gives separate income and expenses details
- Transaction list updates in real time
- Delete transactions

## 🛠 Tech Stack

- HTML
- CSS
- JS

## 🧠 Concepts Practiced

- DOM Manipulation
- Event Handling
- Form Handling
- Array Methods
- Dynamic UI updates
